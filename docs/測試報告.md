## 一、功能測試

> 執行日期 2026-05-31。單元 11/11 通過（Vitest）、E2E 10/10 通過（Playwright，桌機 + 行動各 5）。

| 功能 | 測項 | 工具 | 結果 | 備註 |
|------|------|------|------|------|
| 找吸菸區 | 定位成功 / 拒絕後 fallback 台北中心 | Playwright | ✅ | headless 無定位權限→正確 fallback 台北中心 |
| 找吸菸區 | 附近半徑回傳並依距離排序 | Vitest | ✅ | `data.test.ts` 半徑過濾、距離排序、擴大半徑筆數遞增 |
| 找吸菸區 | 標記與清單一致、導航連結正確 | Playwright | ✅ | 詳情卡導航連結指向 google.com/maps/dir |
| 回報煙味 | 三選一回報狀況、表單顯示 | Playwright | ✅ | 三個描述性選項皆顯示 |
| 回報煙味 | 完成回報流程、成功畫面 | Playwright | ✅ | 送出後顯示「感謝你的回報」 |
| 回報煙味 | 速率限制生效 | Vitest | ✅ | `rateLimit.test.ts` 達上限後擋下 |
| 規則頁 | 內容渲染、法規重點顯示 | Playwright | ✅ | 標題與「哪裡禁菸？」卡片可見 |
| 通用 | 距離計算（geo）單元測試 | Vitest | ✅ | Haversine 距離、格式化、對稱性 |

圖例：✅ 通過｜❌ 失敗｜⏳ 待測

**測試指令**：`npm run test`（單元）、`npm run test:e2e`（端對端）。
