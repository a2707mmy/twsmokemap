## 五、上線除錯紀錄（線上才會出現的問題）

| 問題 | 根因 | 解法 |
|------|------|------|
| Google 地圖顯示「didn't load Google Maps correctly」 | API 金鑰 HTTP 參照網址限制未含正式網域 | 在 Google Cloud 加入 `https://twsmokemap.vercel.app/*` |
| 吸菸區顯示 0 筆 | Vercel 環境變數 `VITE_SUPABASE_URL` 的值被重複貼成三段（`...co https//... https://...`），導致 `ERR_NAME_NOT_RESOLVED` | 重設為單一乾淨網址並 Redeploy |

> 診斷方法：用 Playwright 載入線上版攔截 console 錯誤與失敗請求（`scripts/diag-prod.mjs`），直接看到被打成的錯誤網址，快速定位。
